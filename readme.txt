An executive-level, high-performance WordPress theme ecosystem engineered for Marriage Registries and Institutional data management. Optimized for LiteSpeed servers and the Servinux fintech infrastructure.

== Description ==

**Marriage Registry Master** is a bespoke, enterprise-grade WordPress theme designed by **James P. Friday**, a veteran Full-Stack Web Engineer and the founder of **Servinux**. This theme is architected for modern public institutions that demand a seamless, high-speed user journey without the bloat of traditional multipurpose themes.

As a lead developer for high-traffic platforms, James P. Friday built this theme to solve the "clunky government UI" problem. By utilizing a stateless PHP execution model and a surgical CSS framework, Marriage Registry Master delivers instantaneous load times on **CyberPanel** and **LiteSpeed** environments, ensuring high-concurrency stability during peak registration periods.

It serves as the official design framework for the **Registry Form Pro** ecosystem, providing a unified visual identity from the initial landing page to the final financial settlement via Servinux Checkout.

### Key Technical Features:
* **The Journey UI Engine:** A custom-built, modular step-by-step visualizer that simplifies complex registry processes for citizens.
* **Executive Hero Logic:** High-impact, customizable landing sections designed to establish institutional authority and trust.
* **Fintech-Ready Styling:** Native, pre-configured CSS hooks for a seamless checkout experience via the Servinux and Paystack SDKs.
* **Stateless Performance:** Engineered for low server overhead, utilizing vanilla-lightweight frontend logic for top-tier PageSpeed scores.
* **Institutional Dashboard Components:** Integrated sections for executive messaging, program calendars, and automated status tracking.
* **Developer-First Extendability:** A clean, modular codebase using standard WordPress hooks for easy customization by technical teams.

== Installation ==

1. Upload the `marriage-registry-master` directory to the `/wp-content/themes/` directory.
2. Activate the theme through the 'Appearance > Themes' menu in WordPress.
3. Access the **WordPress Customizer** (Appearance > Customize) to configure brand colors, executive typography, and the Registry Journey steps.
4. Deploy the **Registry Form Pro** plugin to activate the dynamic multi-step application and payment logic.

== Frequently Asked Questions ==

= How is this optimized for CyberPanel/LiteSpeed? =
The theme is designed to work with LiteSpeed’s object caching and ESI (Edge Side Includes) logic, ensuring that institutional data remains fresh while the UI is served at lightning speed from the cache.

= Can I use this for non-marriage registries? =
Absolutely. The "Whitelabel" nature of the customizer allows you to rename "The Registry Process" and all steps to suit any institutional application (e.g., permits, licenses, or school enrollments).

= Does it support the Servinux Checkout SDK? =
Yes. The theme contains dedicated styling logic to ensure the Servinux Drop-in Modal looks professional and integrated with your brand colors.
